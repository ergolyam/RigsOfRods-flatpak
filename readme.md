# RigsOfRods-flatpak

Flatpak version of the game [Rigs of Rods](https://www.rigsofrods.org).

## Installation

```sh
flatpak remote-add --user RigsOfRods https://gmanka-flatpaks.github.io/org.rigsofrods.RigsOfRods/gmanka-flatpaks.flatpakrepo
flatpak install --user RigsOfRods org.rigsofrods.RigsOfRods
```
